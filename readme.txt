Given a discrete data set representing a sample or a population, the calculator calculates the mean, variance, and standard deviation and displays the workflow involved in the calculation.

[https://www.calculator.io/variance-calculator/](https://www.calculator.io/variance-calculator/)

== Usage ==

Add the Variance Calculator shortcode to your page, post or sidebar:

`[ci_variance_calculator]`

Add the following code to your template where you would like the Variance Calculator to appear:

`<?php display_ci_variance_calculator(); ?>`

== Installation ==

1. Upload the Variance Calculator /ci_variance_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Variance Calculator plugin through the "Plugins" menu in WordPress.
